要背的東西
===========

API類
-----

.. important::
    在軟體API方面，程式認的是馬達控制器，這邊要注意一下

REVLib
++++++

- SparkMAX
  
    適用於所有有刷跟無刷的控制器，主要是配NEO系列跟CIM馬達

- SparkFlex
  
    主要是接NEO Vortex跟NEO v2.0的馬達，其他馬達可以透過 Flex Dock來接(有刷馬達就走flex dock)

- Through Bore Encoder
  
    有分 Relative 跟 Absoulte，注意轉換板不要買錯，就是REV的軸編碼器

- Color Sensor V3
  
    毫無反應就是顏色辨識器，不過準度就見仁見智了

- 2m Distance Sensor

    毫無反應就是兩公尺的紅外線距離感測劑

- Blinkin LED Controller

.. important::
    ONE IMP AMONG US: 這個是用WPILib的AddressableLED去控制，不適用REVLib

Phoenix 6
+++++++++

- TalonFX

    內建在Kraken X60/X44裡面的控制器

- TalonFXS

    簡單來說就是CTRE版的SparkMAX

- CANCoder

    CTRE搞得磁編碼器，可以跟TalonFX做深度整合（不過有些功能會需要用到鈔能力就是了）

- CANdi 

    基本上用不到等會要用到再查就好了

- CANdle

    燈條控制器，比REV那個好用超級多

- CANRange

    CTRE的距離感測器，可以測量1cm~3公尺

- Pigeon 2.0

    CTRE的陀螺儀

Phoenix 5
+++++++++

- TalonSRX

    可以接編碼器的有刷馬達控制器

- VictorSPX

    不能接編碼器的有刷馬達控制器

其他有的沒有的東西
------------------

- NavX(AHRS) (以前要插RIO上的版本) - Studica
  
    這是陀螺儀

- NaxX (新的) - StudicaLib
  
    還是陀螺儀

- CANivore

    CTRE搞的可以分CAN迴路跟CAN FD的東西，主要是可以防爆一個全爆的情況，不過在SystemCore出之後可能就會弱掉了
